# 配置工具图标

Jelly 的工具支持灵活的图标配置，您可以配置或创作喜爱的图标，以帮助您快速找到功能。

为了确保显示效果，目前图标只支持矢量风格，不支持 png。

## 内置图标

Jelly 支持系统内置的 SF Symbol 图标，您可填入图标名称添加 SF Symbol。

:::info
由于 Apple 限制，除非下载 SF Symbol App（460MB+），否则目前不能在本地查看全部的 SF Symbol。

若不想下载官方 App，您可以在 Google 搜索 [sf symbols online list](https://www.google.com/search?q=sf+symbols+online+list) 寻找在线预览网站。
:::

## 自定义 SVG

### 支持说明

Jelly 对 SVG 的支持使用 [SVGView](https://github.com/exyte/SVGView) 实现。它不是完整的 SVG 支持，因此，有些 svg（例如内嵌 image 标签的）无法支持。

### 免费 SVG 资源网站推荐

[lucide.dev](https://lucide.dev/icons/)

大量优质的 svg 资源，stroke-width 调到 1.75px 和 SF Symbol 比较搭。

[simpleicons.org](https://simpleicons.org/)

有一些公司 logo

:::info
需要确保下载的 SVG 资源，内部 path 的 stroke color 是 currentColor。这样才能跟随 Jelly 的主题色变化
:::

### 使用 AI 制作 SVG

除直接下载外，您也可以使用 AI 制作 SVG。

:::tip
目前 Claude 以及 Nano Banana Pro 对于 svg 的生成能力比较好。

可以在提示词中加入“SF Symbol 风格”，生成效果会好一些。
:::

#### 编辑 SVG

若您想对 SVG 内部的 path 进行可视化编辑，可以使用 figma 等工具。
