# 语音合成（TTS）服务

## 说明

Jelly 在执行播放音频工具、在收藏中播放语音时，会调用语音合成服务

因为很多语音合成服务支持的语言有限，且不同服务对于不同语言的效果各有优劣（比如有道的语音合成对于英语效果比较好），Jelly 使用了“自动降级”的语音播放策略。

播放语音时，Jelly 会优先尝试调用列表第一个服务。如果语言不支持/调用失败，会依次按顺序调用列表后面的服务。

## 支持服务列表

### 系统语音合成

目前 Jelly 内置仅支持系统语音合成服务。您可以通过[安装插件](/plugin/add-service-plugin)来使用其他语音合成服务。

> 后续会考虑内置支持效果比较好的服务，如豆包、有道等。如果你觉得哪家服务效果比较好，欢迎[反馈给开发者](/about/contact)
