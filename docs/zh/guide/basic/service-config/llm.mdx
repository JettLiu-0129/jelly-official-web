## Q&A

### 什么是 OpenAI 兼容服务？

是指模型服务提供商的 API 接口，兼容 OpenAI 的规范。目前大多数的大模型服务提供商都支持此格式调用。

Jelly 会尝试使用标准的 OpenAI 兼容格式调用这些服务商的 /chat/completions 接口。

你也可以通过选择“自定义 OpenAI”服务并且手动填写 Base URL，来调用任意 OpenAI 兼容服务
