# 收藏功能

Jelly 的收藏功能可能与您常见的应用收藏功能有一些区别。在 Jelly 中，“收藏”不是静态的，它更像一个“历史记录开关”。开启收藏后，Jelly 会从您的划词、调用工具等操作中自动收集和收藏的单词/句子关联的信息，记录到收藏列表历史中。

例如，**当你收藏了一个单词，下次再对包含此单词的句子进行划词时，此句子就会自动作为上下文出现在该单词的划词历史中**，如下图“上下文（自动获取）”效果。

因此，您收藏陌生单词后，第二次遇到时，可以主动对包含该单词的句子进行一次划词操作，以方便 Jelly 记录。

## 查看收藏

<MacScreenShot src="/favorite.png" />

在 设置->收藏 中查看收藏，或在系统托盘 icon 上单击 -> 查看收藏

## 将收藏标记为已掌握

您可以通过单击收藏列表中的学士帽图标，将收藏的单词/句子标记为“已掌握”。标记为“已掌握”后，Jelly 不再会收集该收藏条目关联的信息、不会再记录上下文。
