# 使用 Jelly 辅助学习英语

你好，我是 Jelly 的开发者。我开发 Jelly 的很大一部分初衷，就是为了帮助自己学习英语。我在利用 Jelly 提升英语水平的过程中，也积累了不少经验，分享给大家。

:::warning 注意事项
我有一定英语基础，因此我的学习经验不一定适合所有人。如果你的词汇基础较差，可能不适用我的学习方法。

大概给个参考：我高考英语 140，CET6 560 分，然后上大学之后就没怎么再提升过英语了，上了三年班才又拾起来
:::

## English-first mindset

语言不仅仅是一种表达方式，更是一种思维方式。如果想要用好英语，需要用英语的思维去思考。想要英语表达不卡壳，脑子里直接出现的句子应该直接就是英文的，而不是先出现中文，再翻译成英文。

> 有一定英语基础后，你会发现，当你想表达某个含义的时候，用英语就很自然，但是用中文就比较困难。比如“token”、“vanilla”等词汇，你会发现很难找到一个能准确对应的中文词汇。
>
> 反之，也有用中文表达很简单，用英文表达就很困难的例子。比如“吃苦耐劳”、“缘分”之类的词，很难找到准确的英文对应。

:::details 举一个 ChatGPT 提供的例子
中文：

我随便吃一点就行。

❌ 中式英语直译：

I can eat casually.
（英语里 eat casually 意思是“随便地、不讲礼仪地吃”，完全变味。）

✅ 地道英文说法：

Anything is fine for me.
或
I’m not picky.

为什么？

中文的“随便吃一点”是一种态度（不挑）。

英文需要直接表达“我不挑 / 吃什么都可以”。
:::

一开始词汇量不太够的时候可以查英汉词典，如果有一定基础，一定要用英英词典查词。这样可以帮你形成英文思考方式的闭环。

Jelly Pro 的应用内划词功能就可以帮你很好地在这个场景学习，如果查英英词典的过程中又遇到新的不认识的单词，可以很方便地划词继续查，还能结合上下文解释。

## Output is essential

如果想要用好英语，只输入（reading、listening）是不行的。需要养成使用英语输出（writing、speaking）的好习惯。分享几个我日常用到的输出场景：

### Chat with AI

这个方法非常推荐。具体就是下载一个语音输入软件，如 Wispr Flow，或者直接用系统自带的也行（识别率可能差点）。然后在日常使用 AI 的过程中，全程通过英语和 AI 交流。

如果你是一名程序员，可以在 Vibe Coding 的过程中使用英语。一边工作，一边就可以有很大的提升，效率很高。

这个方法的优点是，你可以即时获得反馈。如果你的发音不准，语音识别软件可能识别不出来。如果意思没有表达准确，AI 就会出现理解偏差，进而回答不准确。这可以非常好地验证，你到底有没有把意思表达清楚。

### Write with Jelly

在工作中，如果有写文档之类的场景，可以用英文写。如果要给其他人看，最后再用大模型翻译成中文就行了。这样就多了一些使用英语写作的机会。

Jelly 在这个场景下也能为你提供很多帮助。

- 如果你不知道写的准不准确，可以选中你写的文本，执行语法纠错工具。

- 如果有一句话你不知道用英文怎么写，可以先用中文写，然后选中这段中文和它的英文上下文，执行 AI 翻译工具（因为有上下文的话 AI 的翻译质量会更好）。之后如果不满意，再执行句子润色工具。

### Chat with people

当然，最好的办法可能还是用英语跟人聊天。但是这个办法对我们 i 人很难行得通。所以就优先推荐前两个办法了。
