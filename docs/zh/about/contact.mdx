为了帮助我们更快定位和解决问题,请在提交 issue 时提供以下信息:

**基础信息:**

- macOS 版本(如:macOS Tahoe 26.0)
- 芯片类型(Intel 或 Apple Silicon)
- Jelly 版本号(可在"关于"中查看)

**问题描述:**

- 问题的具体表现
- 问题发生的频率(偶尔/每次都会)
- 复现步骤(详细描述如何触发该问题)
- 预期行为 vs 实际行为

**环境信息:**

- 是否安装了其他具有划词/剪贴板读写功能的软件（如 PopClip、Bob、Maccy、Cherry Studio 等）
- 系统语言设置

**补充材料:**

- 错误截图或录屏
- 日志（TODO：日志提取方法）
