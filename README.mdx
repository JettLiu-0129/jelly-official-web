<h1 align="center">Jelly</h1>

Jelly 是一个 MacOS 平台的翻译助手。

Jelly 不是开源软件，本仓库主要是 Jelly 的[官方网站](https://jellytranslate.com)的源码。

本仓库未来也会用于开源部分 Jelly 开发过程中的提示词。

您可以在本仓库提交 Jelly 的 Bug。用户反馈和功能建议，请访问官网[联系与反馈](https://jellytranslate.com/about/contact)
